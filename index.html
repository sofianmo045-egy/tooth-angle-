Note: Please refer to the Diagram tab to visualize the points to be surveyed and understand what each element represents in both inputs and calculation results.

Important: This application is a support tool that provides results based on the data provided. Therefore, the most critical factor is measurements accuracy to obtain precise results.

* Best Practices for Accuracy
  • The excavator should be in a level position (Pitch and Roll as close to 0 as possible).
  • It is preferred to extend the excavator arm (Boom and Stick) to the maximum reach to ensure higher accuracy in Heading calculations.
  • You can use Global Coordinates or assume Local Coordinates; however, using Global coordinates is preferred to obtain the Actual Machine Heading.
  • In both cases (Global or Local), the Heading Correction value will remain unaffected.
* Import Survey CSV
  • You can upload points via a CSV file to avoid manual typing errors and save time.
  • The CSV file must contain four columns in this order: (ID, Easting, Northing, Elevation).
  • Points do not need to be in any specific order; the app automatically maps them by ID.
  • Required IDs for auto-mapping: A1, A2, BS, BT, BU, ST, BP, S3, S4, S5, T1.
* Auto-Save & History

The app saves your last inputs automatically. Use the "History" button to review the last 10 calculations.
